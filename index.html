Room Sensor IoT

ESP8266 (NodeMCU Lua) · MQTT · Node-RED · Discord · Docker Compose

End-to-end telemetry pipeline with a real DHT11 sensor, a hardware-free simulator, and a Dockerized local stack (Mosquitto + Node-RED).

  • Publishes retained topics for instant dashboards + non-retained alert topics.
  • Phone-friendly HTTP endpoint on the device (/, /json).
  • Docker quickstart for reproducible PC-side setup.

Telemetry Ingest API

FastAPI · SQLite · Tests · GitHub Actions CI

A small backend service that accepts sensor telemetry over HTTP, validates it, stores the latest reading per device, and exposes query endpoints.

  • Turns MQTT streams into a stable HTTP “current state” API.
  • Rejects malformed payloads (basic junk filtering).
  • Interactive Swagger UI at /docs.
